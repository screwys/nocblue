# sources

- BlueBuild template and GitHub Action for the recipe-based build workflow.
- Universal Blue Silverblue main for the Fedora 44 base image and normal Fedora kernel.
- Zirconium for the niri-focused desktop image direction.
- Noctalia Fedora installation sources through Terra.
- Noctalia SDDM theme from `screwys/noctalia-sddm-theme` pinned at `0b80d9a76f9bb7c871b0ceeb308b57e0b6e6f15a`, installed with nocblue-specific mutable sync paths under `/var/lib/nocblue/sddm`.
- Candy icons from `EliverLara/candy-icons` `master`, installed into `/usr/share/icons/Candy` at image build time.
- Latest niri through yalter's niri COPR.
- Mullvad Browser through Mullvad's official Fedora RPM repository.
