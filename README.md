# nocblue

`nocblue` is a maximal (yes, truly) Fedora Silverblue bootc image for personal use, influenced by [Zirconium](https://github.com/zirconium-dev/zirconium). It ships with [niri](https://github.com/niri-wm/niri), [Noctalia](https://github.com/noctalia-dev/noctalia-shell), 4 natively installed browsers, 30+ flatpak packages, Ghostty, OpenRazer and Proton VPN.

For installation, use a Fedora Silverblue base and run:

```bash
sudo bootc switch ghcr.io/screwys/nocblue:latest
sudo systemctl reboot
```
